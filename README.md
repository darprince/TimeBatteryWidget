# Time Battery Widget

Small Android app project for Android Studio with a home screen widget that shows:

- current time
- battery percentage

## Features

- launcher screen with a preview and one-tap widget pinning
- home screen widget using `TextClock` for live time display
- tap the widget to refresh the battery percentage

## Build

Open the project in Android Studio or run:

```bash
./gradlew :app:assembleDebug
```
